What am I looking at?

Instead of reading a price chart left-to-right, topological data analysis (TDA) turns the last few weeks of returns into a cloud of points and studies its shape. Each day's triple (today, tomorrow, day-after) becomes one point in 3D — the cloud below. Persistent homology counts the cloud's structure: clusters (H0) and loops (H1), and how long they persist as you zoom out. Calm, trending markets make tight simple clouds; turbulent or cyclical markets make stretched clouds with loops. Those numbers feed the model as early-warning "market shape" features that plain returns can't capture.

The idea

Markets move on news, but news is text and models need numbers. This app turns the day's financial headlines into a trading signal in five steps:

  1. 📰 Collect — pull the last few days of headlines per ticker Google News RSS / Yahoo Finance
  2. 🧠 Read — a finance-tuned AI scores each headline positive / negative / neutral FinBERT, with a lexicon fallback
  3. 📐 Shape — recent returns become a 3D point cloud; its shape is measured topological data analysis
  4. 🎲 Predict — sentiment + momentum + shape → P(up tomorrow) per ticker rule blend / logistic / XGBoost
  5. 📊 Test — simulate trading that signal historically, minus costs, vs. buy & hold backtester

Demo mode generates a self-consistent synthetic market (headlines and prices, offline, seeded) so you can explore instantly. Live mode pulls real headlines and prices. No look-ahead bias: signals formed today are only ever evaluated on tomorrow's returns.

⚠️ Educational project. Backtests on short windows are noisy; past performance guarantees nothing.
